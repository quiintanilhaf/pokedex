# 🧩 Pokédex Interativa

Uma Pokédex moderna e responsiva feita com **HTML, CSS e JavaScript**, consumindo dados da API oficial Pokémon: a PokéAPI.

O projeto permite visualizar Pokémons em cards interativos e acessar uma página com detalhes completos de cada Pokémon.

---

# 🚀 Funcionalidades

✅ Listagem dinâmica de Pokémons usando API
✅ Cards interativos
✅ Página individual para cada Pokémon
✅ Cores baseadas no tipo do Pokémon
✅ Layout responsivo
✅ Efeitos visuais e animações
✅ Menu lateral de categorias
✅ Navegação entre páginas

---

# 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* PokéAPI
